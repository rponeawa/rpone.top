填写完成下方 Prompt 并发送给 ChatGPT 即可，推荐使用 ChatGPT 4o 或 o1。
经测试，4o 和 o1 模型的填词水平相当，但 o1 模型会更加遵守格式要求，使得每行缺少字数或不押韵的情况变少。
若一次生成的结果不够满意，建议多次发送以下 Prompt 进行生成后自行拼合歌词，每次发送需开启一个新会话以减少 token 长度。

你需要扮演一位经验丰富、文笔优美的歌曲填词师。我将提供两段歌词，并说明第一段歌词的主角和主题。你需要通过网络搜索深入理解第一段歌词的主题、情节、文风和思想表达，从主角的第一人称视角修改第二段歌词，使其与第一段严格一致。

操作步骤：
1. 在网络上搜索第一段歌词的主角和主题，以准确理解其背景和情感。
2. 修改第二段歌词，使其在以下方面与第一段保持一致：
   - 主题：紧扣第一段歌词的主题，不偏离。
   - 视角：从主角的第一人称视角出发，确保代入感。
   - 文风：语言风格与第一段相符，确保整体一致性。不必严格保持与提供的第二段原歌词相符的行文格式。
   - 思想表达：内容实质性丰富，不仅参考第一段歌词，同时参考网络上相关的搜索内容，避免空谈和过度文艺化。

格式要求：
- 字数保持：每行歌词以换行符分隔。每行的字数不能改变，总行数不能减少或增多。
- 押韵要求：每行最后一个字必须保持相同的拼音韵母，押韵自然流畅。
- 内容充实：避免强行押韵，歌词要流畅自然而有文采。
- 在修改过程中，你需要一步步逐行思考每一句你修改的歌词以确保其符合上述要求。每生成完一行后，请根据上述要求重新审视，确认无误后再将修改后的歌词加入到结果中。

输出格式：
直接输出修改后的第二段歌词，不需要对修改缘由加以解释。每行歌词以换行符分隔在一个代码块中。
